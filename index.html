Matthew Sprague's MIDI Generator

Project Objective

The goal of this project is to create a Python-based tool that reads a custom DSL to author music and generates corresponding MIDI files. This allows users to easily script musical compositions with commands for tempo, instruments, and notes.

GitHub Repository

Check out my repo for the MIDI generator: GitHub: chords2midi

Sample DSL Input

                tempo 120
                instrument piano
                note C4 1
                note E4 1
                note G4 1
                chord Cmaj 2
                rest 1
                instrument guitar
                chord Gmaj 2
                note D4 1
